List of compatible HTML widgets

This page lists some HTML widgets that are compatible with Crosstalk.

(To add a widget, please submit a PR with modifications to this page.)

Plotly

Plotly is a general purpose plotting library that outputs interactive web graphics.

devtools::install_github("ropensci/plotly")

Leaflet

Leaflet lets you easily generate interactive maps using R.

devtools::install_github("rstudio/leaflet")

DT

DT is a highly customizable data table library.

devtools::install_github("rstudio/DT")
